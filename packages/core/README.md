# @doom-ui/core

Core functionality for the Doom UI component library.

## Installation

```bash
npm install @doom-ui/core
```

## Usage

```typescript
// tailwind.config.js
import { doomUI } from "@doom-ui/core";

module.exports = {
  // ...
  plugins: [
    doomUI({
      // your config here
    }),
  ],
};
```

## Features

- Theme configuration system
- Dark mode support
- CSS variable system
- Utility functions
